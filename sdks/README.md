# SDKs

Notary SDK families, route manifests, and contract tests.

- `sdkwork-notary-app-sdk/`: generated app API TypeScript SDK and composed facade
- `sdkwork-notary-backend-sdk/`: generated backend API TypeScript SDK and composed facade
- `_route-manifests/`: canonical HTTP route manifests for Rust route crates
- `test/`: workspace contract tests executed by `pnpm test:contracts`
