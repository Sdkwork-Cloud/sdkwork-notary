# Crates

Rust workspace crates for the notary domain library.

| Crate | Role |
| --- | --- |
| `sdkwork-notary-case-contract` | Domain contract types and service port traits |
| `sdkwork-notary-case-service` | Runtime orchestration and operation dispatch |
| `sdkwork-notary-case-repository-sqlx` | SQLx persistence via `sdkwork-database` |
| `sdkwork-routes-notary-http-auth` | Shared dual-token web framework layer |
| `sdkwork-routes-notary-app-api` | App API Axum route crate |
| `sdkwork-routes-notary-backend-api` | Backend API Axum route crate |
